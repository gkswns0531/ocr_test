# True Visual 40건 케이스 분석

SDS-KoPub-VDR-Benchmark 600개 QA 중 `type=visual` 161건을 정성 전수 검토하여 **true_visual 58건** 분류 →
GT 페이지에 image/chart region이 존재하는 **40건**에 대한 상세 분석.

- **Corpus**: 40,781 pages
- **Regions**: 21,052
- **Caption Model**: 4o
- **Embedding Model (본 분석 기준)**: Qwen3-VL-Embedding-8B-FP8 (dim=4096)
